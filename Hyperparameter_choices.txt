| Component                 | Choice / Value                                   | Justification                                                        |
| ------------------------- | ------------------------------------------------ | -------------------------------------------------------------------- |
| **Model Architecture**    | `InceptionV3 (pretrained=True, aux_logits=True)` | Leverages ImageNet-learned features + auxiliary branch for stability |
| **Input Size**            | `299 × 299`                                      | Required input size for InceptionV3                                  |
| **Optimizer**             | `Adam (lr=1e-4)`                                 | Adaptive learning rate, fast convergence for transfer learning       |
| **Loss Function**         | `CrossEntropyLoss()`                             | Standard choice for binary/multi-class classification                |
| **Learning Rate**         | `1e-4`                                           | Stable training for fine-tuning deep networks                        |
| **Batch Size**            | `32`                                             | Balanced memory and convergence speed                                |
| **Epochs**                | `Max: 20` (with early stopping)                  | Avoid overfitting by monitoring validation AUROC                     |
| **Early Stopping**        | `Patience = 5` based on validation AUROC         | Stops training if no improvement for 5 epochs                        |
| **Data Augmentation**     | `HorizontalFlip, Rotation(10°)`                  | Improves generalization with image variability                       |
| **Sampler**               | `WeightedRandomSampler`                          | Oversampling of minority class to handle class imbalance             |
| **Validation Strategy**   | `5-Fold Stratified K-Fold`                       | More robust, less biased evaluation on full dataset                  |
| **Evaluation Metrics**    | `Accuracy, F1-Score, AUROC`                      | F1 handles imbalance, AUROC reflects classifier confidence           |
| **Final Model Selection** | Based on best **validation AUROC**               | AUROC is threshold-independent and more informative than accuracy    |
